## DESCRIERE GENERALĂ

Campaia este o platformă bazată pe Inteligență Artificială care ajută ONG-uri, inițiative comunitare și proiecte locale (donare de sânge, reciclare, voluntariat etc.) să creeze campanii digitale eficiente pentru social media, în doar câteva minute.

Utilizatorul introduce câteva detalii despre campania sa (scop, public țintă, locație), iar platforma generează automat:
- Scripturi video pentru TikTok / social media
- Imagini promoționale
- Videoclipuri gata de publicat
- Hashtag-uri și strategii de distribuție
- Recomandări pentru audiența țintă

## TECH STACK IMPLEMENTAT

### Backend (campaia-engine)
- Framework: FastAPI (Python 3.12)
- Bază de date: PostgreSQL 16 (async cu asyncpg)
- ORM: SQLAlchemy 2.0 (async)
- Migrații: Alembic
- Cache: Redis 7
- Autentificare: JWT (python-jose) + Google OAuth
- Package manager: Poetry

### Frontend (campaia-platform)
- Framework: React 19 cu TypeScript
- Build tool: Vite
- Styling: Tailwind CSS 4
- Auth: @react-oauth/google
- Mobile: Capacitor (suport pentru Android)

### AI & Integrări externe
- Ollama — LLM local pentru generare de text (scripturi, descrieri, hashtag-uri)
- Kling AI — API extern pentru generare video (text-to-video, image-to-video)
- AWS S3 — Stocare obiecte (LocalStack pentru dev)
- Stripe — Procesare plăți (sistem de tokeni)
- TikTok Ads API — Publicare campanii pe TikTok

### Infrastructură
- Docker Compose — Orchestrare locală (PostgreSQL, Redis, Backend, Frontend, LocalStack)
- Health checks configurate pe toate serviciile

## FUNCȚIONALITĂȚI IMPLEMENTATE (DETALIAT)
====================================================================

### 1. AUTENTIFICARE & MANAGEMENT UTILIZATORI
- Login cu email + parolă (cu hash bcrypt)
- Login cu Google OAuth (înregistrare/autentificare automată)
- JWT tokens pentru sesiuni persistente (utilizatorul rămâne logat și după refresh)
- Redirect automat la dashboard după login
- Route-uri API protejate cu JWT
- Pagini: Signin, Signup, Landing Page
- Tipuri de utilizator: INDIVIDUAL și BUSINESS
- Profil complet cu câmpuri business: CUI, Reg. Com., adresă, oraș, județ, țară
- Flag de "profile completed"

### 2. SISTEM DE CAMPANII
- CRUD complet pentru campanii (creare, citire, actualizare, ștergere)
- Status tracking: DRAFT → ACTIVE → PAUSED → COMPLETED / CANCELLED
- Tranziții de stare validate (activate, pause, resume)
- Câmpuri: nume campanie, URL destinație, buget, durată, descriere produs, oraș (city), coordonate (lat, lng)
- Script AI atașat campaniei
- Tracking tokeni consumați per campanie
- Paginare pe lista de campanii
- Filtrare după status
- Pagini dashboard: Overview, Campaign Details, New Campaign, History

### 3. GENERARE CONȚINUT AI (via Ollama)
- Generare scripturi video (pentru TikTok) — multiple variante (5 by default)
- Suport pentru tonuri: profesional, casual, viral, funny
- Generare în limba utilizatorului (detecție automată RO/EN)
- Generare hashtag-uri relevante
- Sugestii de audiență țintă (demographic recommendations)
- Generare descrieri marketing "punchline"
- Generare prompt-uri cinematice pentru Kling AI
- Verificare disponibilitate Ollama

### 4. GENERARE VIDEO AI (via Kling AI)
- Generare video din text (text-to-video)
- Generare video din imagine (image-to-video)
- Workflow async complet: creare job → apel Kling API → polling → download → upload S3 → update DB
- Calități video: STANDARD, PROFESSIONAL
- Durată video: SHORT (5s), MEDIUM (10s)
- Aspect ratio configurabil
- Upload manual de video-uri (din galerie/cameră)
- Ștergere video (din DB + S3)
- Editare titlu video
- My Videos — galerie personală cu toate videourile utilizatorului
- Community Feed — feed public cu videourile tuturor utilizatorilor
- VideoGenerator component — interfață de generare
- VideoPreview modal — preview video cu detalii
- VideoThumbnail component — thumbnail vizual
- VideoUpload component — upload de video

### 5. SISTEM DE TOKENI & PLĂȚI (Stripe)
- Wallet per utilizator (balanță + lifetime purchased)
- Tranzacții: PURCHASE, SPEND, REFUND
- Acțiuni token: PURCHASE, BONUS, SCRIPT, IMAGE, VIDEO
- Pachete de tokeni (Stripe checkout):
  • Starter Pack: 100 tokeni — 29 RON
  • Standard Pack: 300 tokeni — 79 RON
  • Pro Pack: 700 tokeni — 149 RON
  • Business Pack: 1500 tokeni — 299 RON
- Checkout Stripe cu redirect (success/cancel)
- Webhook Stripe pentru confirmare plată
- Mock mode când Stripe key lipsește
- Pagini: BuyTokens, PaymentSuccess, PaymentCancel
- TokenBalance component — afișare balanță
- Billing page — detalii billing

### 6. FACTURARE
- Generare automată facturi după plata Stripe
- Număr factură unic: CMP-YYYY-XXXX
- PDF generat server-side (PDFService)
- Upload PDF pe S3
- Download factură prin URL presemnat S3
- Trimitere factură pe email
- Istoric facturi per utilizator
- Suport tip factură și status
- Componente: InvoiceList, InvoiceDownloadButton

### 7. AUDIENCE TARGETING
- Model AudienceTarget atașat campaniei
- Selector demografic (vârstă min/max, gen)
- Selector geografic (locații)
- Interese disponibile (lista statică RO/EN)
- Validare criterii targeting
- SimpleTargetingSelector — interfață simplificată de targeting
- DemographicSelector, GeoSelector — componente dedicate

### 8. INTEGRARE TIKTOK ADS (+ Sandbox)
- Ghid rapid: docs/TIKTOK_SANDBOX.md (env, flux publicare, verificare Ads Manager)

### 8b. INTEGRARE TIKTOK ADS (detaliu)
- Client TikTok API complet implementat
- Publisher TikTok Ads (creare campanie → ad group → ad)
- API routes pentru TikTok
- Câmpuri pe campanie: tiktok_ad_id, tiktok_campaign_id, tiktok_adgroup_id
- Suport sandbox/production
- Pagina Integrations pe dashboard

### 9. ANALYTICS DASHBOARD
- Pagina Analytics cu grafice de performanță
- Vizualizare: views, clicks, CPA, shares
- Selectare timeframe pentru date
- Comparare campanii (reach și ROAS)
- Grafice: line, area, bar charts

### 10. HARTA COMUNITARĂ (WOW Factor)
- Pagina Dashboard → Community Map
- Harta interactivă (Leaflet) cu toți markerii campaniilor active
- Vizualizare impact global: Reach Estimat Total, Număr campanii active
- Filtre avansate: după categorie (Donare de sânge, Reciclare, etc.) și după oraș
- Pop-up-uri detaliate per campanie: Titlu, Categorie, Target, Reach, Video Link
- Posibilitatea de a alege orașul de publicare per campanie (București, Cluj, Timișoara, Iași, Constanța, Brașov, etc.)
- Salvarea coordonatelor (lat, lng) în baza de date pentru plasare precisă pe hartă
- Filtrare dinamică pe hartă în funcție de orașele selectate de utilizatori
