## DESCRIERE GENERALĂ

Campaia este o platformă bazată pe Inteligență Artificială care ajută ONG-uri, inițiative comunitare și proiecte locale (donare de sânge, reciclare, voluntariat etc.) să creeze campanii digitale eficiente pentru social media, în doar câteva minute.

Utilizatorul introduce câteva detalii despre campania sa (scop, public țintă, locație), iar platforma generează automat:
- Scripturi video pentru TikTok / social media
- Imagini promoționale
- Videoclipuri gata de publicat
- Hashtag-uri și strategii de distribuție
- Recomandări pentru audiența țintă

## TECH STACK IMPLEMENTAT

### Backend (campaia-engine)
- Framework: FastAPI (Python 3.12)
- Bază de date: PostgreSQL 16 (async cu asyncpg)
- ORM: SQLAlchemy 2.0 (async)
- Migrații: Alembic
- Cache: Redis 7
- Autentificare: JWT (python-jose) + Google OAuth
- Package manager: Poetry

### Frontend (campaia-platform)
- Framework: React 19 cu TypeScript
- Build tool: Vite
- Styling: Tailwind CSS 4
- Auth: @react-oauth/google
- Mobile: Capacitor (suport pentru Android)

### AI & Integrări externe
- Ollama — LLM local pentru generare de text (scripturi, descrieri, hashtag-uri)
- Kling AI — API extern pentru generare video (text-to-video, image-to-video)
- AWS S3 — Stocare obiecte (LocalStack pentru dev)
- Stripe — Procesare plăți (sistem de tokeni)
- TikTok Ads API — Publicare campanii (sandbox sau production)

### Infrastructură
- Docker Compose — PostgreSQL, Redis, Backend, Frontend, LocalStack
- Health checks pe servicii
- Pornire locală: ./scripts/dev-start.sh (sau docker compose up -d)
- Regulă Cursor (.cursor/rules/dev-workflow.mdc): după modificări utile — Docker live + commit-uri mici

## FUNCȚIONALITĂȚI IMPLEMENTATE (DETALIAT)
====================================================================

### 1. AUTENTIFICARE & MANAGEMENT UTILIZATORI
- Login email + parolă (bcrypt), Google OAuth, JWT persistent
- Route-uri API protejate; redirect dashboard după login
- Tipuri: INDIVIDUAL, BUSINESS (CUI, Reg. Com., adresă, oraș, județ, țară)
- Flag profile completed

### 2. SISTEM DE CAMPANII
- CRUD complet; status: DRAFT → ACTIVE → PAUSED → COMPLETED / CANCELLED
- Câmpuri: nume, URL destinație, buget, durată, descriere, city, lat, lng
- Script AI, tokeni per campanie, paginare, filtrare status
- Pagini: Overview, Campaign Details, New Campaign, History

### 3. GENERARE CONȚINUT AI (Ollama)
- Scripturi TikTok (5 variante default), tonuri (profesional, casual, viral, funny)
- RO/EN, hashtag-uri, audiență, punchline, prompturi Kling
- Health Ollama

### 4. GENERARE VIDEO (Kling + S3)
- Text-to-video, image-to-video, job async, upload S3, galerie
- My Videos, Community Feed; VideoGenerator, Gallery, Upload, Preview

### 5. TOKENI & STRIPE
- Wallet, pachete (Starter 100/29 RON … Business 1500/299 RON)
- Checkout, webhook, mock fără key; BuyTokens, Billing

### 6. FACTURARE
- Facturi CMP-YYYY-XXXX, PDF, S3, email, InvoiceList

### 7. AUDIENCE TARGETING
- AudienceTarget; DemographicSelector, GeoSelector, SimpleTargetingSelector

### 8. TIKTOK ADS (SANDBOX / PRODUCTION)
- Client + publisher: upload video URL → campanie TikTok → ad group → ad
- API: /api/v1/tiktok/status, /publish, pause/resume, campaigns list, metrics (restrict)
- Campanie: tiktok_campaign_id, tiktok_adgroup_id, tiktok_ad_id
- Publicare din Campaign Details: necesită video cu URL public (S3); buton activ doar cu video_url
- Env backend (.env + docker-compose): TIKTOK_ACCESS_TOKEN, TIKTOK_ADVERTISER_ID, TIKTOK_APP_ID, TIKTOK_APP_SECRET, TIKTOK_ENVIRONMENT (sandbox|production), TIKTOK_METRICS_ADMIN_EMAILS (listă emailuri pentru GET /tiktok/metrics)
- Documentație flux sandbox: docs/TIKTOK_SANDBOX.md
- Pagina Integrations: status cont, Campaign Explorer (campanii din API), checklist publicare sandbox; metrici globale doar pentru admini (restul vede mesaj, fără erori în UI)

### 9. ANALYTICS DASHBOARD
- Recharts: views, clicks, CPA, shares; timeframe; comparare campanii (reach / ROAS)

### 10. COMMUNITY MAP
- Leaflet, markere campanii active, filtre categorie/oraș, lat/lng, pop-up-uri

### 11. PAGINI FRONTEND
- Landing, Concept, Privacy, Terms, Signin/Signup
- Dashboard: Overview, New Campaign, Campaign Details, My Videos, Analytics, Billing, Buy Tokens, History, Integrations, Profile, Settings, Community Map

### 12. COMPONENTE UI (esențial)

### 13. SERVICII FRONTEND

### 14. MODELE DB
- User, Campaign, Wallet, TokenTransaction, VideoGeneration, AudienceTarget, Invoice

### 15. DEVOPS
