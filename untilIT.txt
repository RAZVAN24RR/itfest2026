## DESCRIERE GENERALĂ

Campaia este o platformă AI pentru ONG-uri și inițiative locale (donare sânge,
hackathon, voluntariat, maraton, reciclare etc.) care creează rapid campanii
digitale pentru TikTok: scripturi, video, targeting, publicare sandbox și
instrumente de planificare (scheduler + agent Gemini).

Track hackathon: **Community Problems** (ITFest 2026).

## TECH STACK

### Backend (campaia-engine)
- FastAPI, Python 3.12, PostgreSQL 16 (asyncpg), SQLAlchemy 2.0, Alembic, Redis 7
- JWT + Google OAuth, Poetry, Docker

### Frontend (campaia-platform)
- React 19, TypeScript, Vite, Tailwind CSS 4, Leaflet (hartă), Framer Motion (unde e folosit)
- Capacitor (Android), @react-oauth/google

### AI & integrări
- **Ollama** — două „niveluri” în UI: **Llama 3.2** (1× tokeni) și **DeepSeek R1 8B** (2× tokeni);
  timeout generos (ex. 300s) pentru modelul mare.
- **Google Gemini** — **Campaia Agent** (chat marketing / planificare), model implicit
  **gemini-2.5-flash** + fallback în cod; cheie: GEMINI_API_KEY; backend cu env_file .env.
- **Kling AI** — video cloud; Runway/Pika — stub/fallback UX
- **ModelScope local** — video pe GPU (STABLE_VIDEO_URL, serviciu separat)
- Stripe, LocalStack S3, TikTok Ads API (sandbox)

### Infrastructură
- Docker Compose: db, redis, backend (host network opțional), frontend, localstack
- După modificări pe model DB: **alembic upgrade head** (ex. coloane noi pe campaigns)

## FUNCȚIONALITĂȚI IMPLEMENTATE (DETALIAT)
====================================================================

### 1. AUTENTIFICARE & UTILIZATOR
- Email/parolă, Google OAuth, JWT, tipuri INDIVIDUAL / BUSINESS

### 2. CAMPANII
- CRUD, status DRAFT → ACTIVE → PAUSED → COMPLETED / CANCELLED
- Câmpuri: nume, URL, buget, durată, descriere, **event_type** (tip eveniment comunitar),
  **city, lat, lng** (hartă + creare campanie)
- Script AI, TikTok IDs pe campanie, paginare

### 3. TIPURI EVENIMENT COMUNITAR (event_type)
- Selectabile la **Campanie nouă** (wizard): donare sânge, hackathon, voluntariat,
  reciclare, adunare comunitară, fonduri, educație, sănătate, sport, cultură,
  animale, urgențe, **maraton**, etc.
- Afișate în **Overview** și **Detalii campanie**; **Campaia Agent** le cunoaște în prompt

### 4. SCHEDULER CAMPANII
- Model **CampaignSchedule**: zile săptămână, interval orar, fus, enable/disable
- API **/api/v1/scheduler/{campaignId}** (GET/POST/DELETE)
- UI în **Campaign Details**: program automat (zile + ore)

### 5. GENERARE AI (Ollama) — DUAL MODEL
- Request-uri AI cu **ai_model**: **llama** | **deepseek**
- Cost tokeni: baza ×1 (llama) sau ×2 (deepseek) pe: script, hashtag, audiență,
  descriere marketing, prompt Kling
- UI selector model în pasul de generare (New Campaign)

### 6. CAMPAIA AGENT (GEMINI)
- Tab **Campaia Agent**: chat cu context Campaia (pagini, tokens, video, scheduler,
  tipuri eveniment, TikTok sandbox)
- API **POST /api/v1/agent/chat**; erori afișate clar (cheie, model, etc.)
- Config: GEMINI_API_KEY, GEMINI_MODEL (default gemini-2.5-flash)

### 7. VIDEO (Kling + local + fallback)
- Galerie, stiluri, tokeni per provider; local ModelScope dacă STABLE_VIDEO_URL e up

### 8. TOKENI & STRIPE
- Wallet, pachete, Checkout, Billing, Buy Tokens

### 9. FACTURARE
- PDF facturi; **CAMPAIA SRL** (nu DEVDELTA) în PDF și email
- CMP-YYYY-XXXX, S3, listă facturi

### 10. TARGETING
- Țări + orașe (RO multi-oras), SimpleTargetingSelector, TikTok mapping

### 11. TIKTOK ADS
- Publicare sandbox, pause/resume, Integrations (admin), metrici restricționate

### 12. COMMUNITY MAP (HARTĂ)
- **GET /api/v1/campaigns/map** — markere pentru toate campaniile (fără auth pe hartă)
- Leaflet, filtre categorie + oraș
- Iconițe pe **event_type**; popup: **Impressions, Clicks, CTR, Shares, Spend (RON)**
  — date din DB (**stats_***) sau calculate determinist dacă lipsesc
- Carduri sus: număr campanii filtrate, total impressions + clicks

### 13. METRICI DEMO PE CAMPANII (TikTok-style)
- Migrație **c3d4e5f6a7b8**: campaigns.**stats_impressions**, **stats_clicks**,
  **stats_shares**, **stats_spend_ron**
- **Obligatoriu** după deploy: `alembic upgrade head` — altfel listarea campaniilor
  crapă (coloane lipsă)
- Script seed (opțional): **campaia-engine/scripts/seed_community_map_demo.py**
  — campanii demo pe cont **razvanandreipasaran@gmail.com**, categorii + orașe diverse

### 14. ANALYTICS (pagină dashboard)
- Legată corect în Dashboard (în trecut randa gol)
- **Fără Recharts** pe această pagină (evită crash React 19 / chunk)
- Interval **calendar De la / Până la** + „Ultimele 30 zile”
- Date din **getMapMarkers()**: agregat impressions, clicks, CTR, spend; tabel campanii;
  bare pe tip; fără secțiune „For judges”
- Mesaj: agregat din campaniile de pe hartă (stil TikTok Ads demo)

### 15. PAGINI DASHBOARD
- Overview, New Campaign, Details, My Videos, **Analytics**, **Community Map**,
  **Campaia Agent**, Billing, Buy Tokens, Profile, Settings, Integrations (admin)

### 16. MODELE DB (esențial)
- User, Campaign (+ event_type, lat, lng, city, **stats_***), CampaignSchedule,
  Wallet, TokenTransaction, VideoGeneration, AudienceTarget, Invoice

### 17. API URI importante
- Prefix **/api/v1/** pentru agent, scheduler, campaigns, etc.
- Frontend: VITE_API_URL către backend (ex. http://localhost:8000)
